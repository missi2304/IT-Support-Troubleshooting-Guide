# IT Support Troubleshooting Guide

## Common Issues and Fixes

### 1. Computer Not Connecting to Internet
- Check IP address (ipconfig /all)
- Restart router
- Flush DNS: ipconfig /flushdns

### 2. Slow Computer
- Check Task Manager
- Disable startup apps
- Run antivirus scan

### 3. Printer Not Working
- Restart print spooler
- Reinstall drivers
